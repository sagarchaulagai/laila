Also write a user defined function to calculate the direct DFT of given signal x[n]. The function may
have the following syntax:
function y = dft_f(x)
    N = length(x);         
    y = zeros(1, N);       
    for k = 0:N-1          
        for n = 0:N-1     
            y(k+1) = y(k+1) + x(n+1) * exp(-1i*2*pi*k*n/N);
        end
    end
end
